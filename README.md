# Proyecto MasNouIMC

**MasNouIMC** es una aplicación para gestionar torneos de ajedrez en formato MasNou.

## Flujo del Programa

Para empezar un torneo, sigue estos pasos:

1. Añadir a los jugadores.
2. Ir a **Ajustes → Cambiar Parámetros** y establecer:
   - Número de mesas del torneo
   - Tiempo de duración de las partidas
3. Ir a **Ajustes → Iniciar** para realizar los emparejamientos automáticamente.

## Requisitos

- JDK 20
- Kotlin
- Gradle

## Estado Actual de la Aplicación

La aplicación todavía está en desarrollo. Actualmente se puede:  
- Añadir jugadores  
- Gestionar emparejamientos básicos  
- Registrar puntuaciones
